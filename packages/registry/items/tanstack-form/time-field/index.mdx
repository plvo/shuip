`TimeField` is a time-only picker that encapsulates TanStack Form's field management with shadcn/ui's design system. It binds to a `string` in the native `HH:mm` 24-hour format that the browser's `<input type="time">` produces — no `Date` object ceremony, no popover, no third-party time wheel.

It reads the surrounding field via `useFieldContext`, so you compose it inside a `<form.AppField>` rather than passing a `form` instance down by prop.

For date-only or date-and-time inputs, use the dedicated date items instead.

#### Built-in features

- **Native browser picker**: leverages `<input type="time">` — zero JS overhead, full accessibility, locale-aware formatting
- **Context-bound field state**: reads the field via `useFieldContext` — no prop drilling
- **Type-safe field names**: `name` autocompletes from your `defaultValues` on `<form.AppField>`
- **Range constraints**: pass `min` / `max` (as `HH:mm` strings) through `props` to restrict the selectable range
- **Step control**: pass `step={60}` (minute, default) or `step={1}` (seconds) through `props`

## Setup

Field components are bound via React context. In your project, create `lib/form.ts` once:

```tsx
// lib/form.ts
import { createFormHook } from '@tanstack/react-form';
import { fieldContext, formContext } from '@/components/ui/shuip/tanstack-form/form-context';
import { SubmitButton } from '@/components/ui/shuip/tanstack-form/submit-button';
import { TimeField } from '@/components/ui/shuip/tanstack-form/time-field';

export const { useAppForm } = createFormHook({
  fieldContext,
  formContext,
  fieldComponents: { TimeField },
  formComponents: { SubmitButton },
});
```

See the [`form-context`](/docs/tanstack-form/form-context) item for details.

## Usage

The value is a `string` in `HH:mm` format (or empty string when unset), matching the native input semantics. You do not need to parse or format anything — store it as-is, validate it as a string, render it as a string.

```tsx
import { useAppForm } from '@/lib/form';

const form = useAppForm({
  defaultValues: { meetingTime: '' },
  onSubmit: async ({ value }) => {
    await save(value);
  },
});

<form.AppField
  name='meetingTime'
  validators={{
    onChange: ({ value }) => (!value ? 'Required' : undefined),
  }}
  children={(field) => <field.TimeField label='Meeting time' />}
/>
```

## Constraining the range

`min`, `max`, and `step` are native input attributes — pass them through `props`:

```tsx
<form.AppField
  name='appointment'
  validators={{
    onChange: ({ value }) =>
      value < '09:00' || value > '18:00' ? 'Outside office hours' : undefined,
  }}
  children={(field) => (
    <field.TimeField
      label='Appointment'
      props={{ min: '09:00', max: '18:00' }}
    />
  )}
/>
```

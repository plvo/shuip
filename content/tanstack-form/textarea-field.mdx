## Built-in features

- **InputGroup integration**: Uses shadcn InputGroup for clean UI
- **Tooltip support**: InfoIcon with contextual help at bottom-right
- **Type-safe**: Full TypeScript support with TanStack Form
- **Validation**: Sync and async validation support
- **Auto-resize**: Supports textarea props like rows, maxLength

## Common use cases

<CodeHighlight code={`const form = useForm({
  defaultValues: {
    bio: '',
    message: '',
    notes: '',
  },
  onSubmit: async ({ value }) => {
    await saveData(value)
  },
})

// Basic textarea
<TextareaField
  form={form}
  name='bio'
  label='Biography'
  description='Tell us about yourself'
  props={{ rows: 4 }}
/>

// With validation
<TextareaField
  form={form}
  name='message'
  label='Message'
  props={{ rows: 6, maxLength: 500 }}
  formProps={{
    validators: {
      onChange: ({ value }) => {
        if (value.length < 10) return 'Message too short (min 10 characters)'
        if (value.length > 500) return 'Message too long (max 500 characters)'
        return undefined
      }
    }
  }}
/>

// With tooltip
<TextareaField
  form={form}
  name='notes'
  label='Notes'
  tooltip='These notes will be visible to your team members'
  props={{ rows: 5, placeholder: 'Add your notes here...' }}
/>`} language='tsx' />

## Built-in features

- **Type-safe options**: Options as `Array<{label: string; value: string}>`
- **Type-safe field names**: Autocomplete with `DeepKeys<TFormData>`
- **Validation**: Sync and async validation support
- **Accessible**: Proper label associations and ARIA attributes

## Common use cases

<CodeHighlight code={`const form = useForm({
  defaultValues: {
    plan: '',
    paymentMethod: '',
    shippingSpeed: '',
  },
  onSubmit: async ({ value }) => {
    await saveData(value)
  },
})

// Basic radio group
<RadioField
  form={form}
  name='plan'
  options={[
    { label: 'Free', value: 'free' },
    { label: 'Pro', value: 'pro' },
    { label: 'Enterprise', value: 'enterprise' },
  ]}
  label='Subscription Plan'
  description='Choose your subscription plan'
/>

// Required selection
<RadioField
  form={form}
  name='paymentMethod'
  options={[
    { label: 'Credit Card', value: 'card' },
    { label: 'PayPal', value: 'paypal' },
    { label: 'Bank Transfer', value: 'bank' },
  ]}
  label='Payment Method'
  formProps={{
    validators: {
      onChange: ({ value }) =>
        !value ? 'Please select a payment method' : undefined
    }
  }}
/>

// With conditional validation
<RadioField
  form={form}
  name='shippingSpeed'
  options={[
    { label: 'Standard (5-7 days)', value: 'standard' },
    { label: 'Express (2-3 days)', value: 'express' },
    { label: 'Overnight', value: 'overnight' },
  ]}
  label='Shipping Speed'
  formProps={{
    validators: {
      onChange: ({ value }) => {
        const total = form.getFieldValue('total')
        if (value === 'overnight' && total < 50) {
          return 'Overnight shipping requires minimum order of $50'
        }
        return undefined
      }
    }
  }}
/>`} language='tsx' />

## Built-in features

- **Visibility toggle**: Eye/EyeOff button to show/hide password
- **InputGroup integration**: Uses shadcn InputGroup for clean UI
- **Tooltip support**: InfoIcon with contextual help
- **Type-safe**: Full TypeScript support with TanStack Form
- **Validation**: Sync and async validation support

## Common use cases

<CodeHighlight code={`const form = useForm({
  defaultValues: {
    password: '',
    confirmPassword: '',
  },
  onSubmit: async ({ value }) => {
    await savePassword(value)
  },
})

// Basic password field
<PasswordField
  form={form}
  name='password'
  label='Password'
  description='Must be at least 8 characters'
/>

// With validation
<PasswordField
  form={form}
  name='password'
  label='Password'
  formProps={{
    validators: {
      onChange: ({ value }) => {
        if (value.length < 8) return 'Must be at least 8 characters'
        if (!/[A-Z]/.test(value)) return 'Must contain uppercase letter'
        if (!/[0-9]/.test(value)) return 'Must contain a number'
        return undefined
      }
    }
  }}
/>

// With tooltip
<PasswordField
  form={form}
  name='password'
  label='Password'
  tooltip='Use a strong password with letters, numbers, and symbols'
  formProps={{
    validators: {
      onChange: ({ value }) =>
        value.length < 8 ? 'Too short' : undefined
    }
  }}
/>

// Confirm password
<PasswordField
  form={form}
  name='confirmPassword'
  label='Confirm Password'
  formProps={{
    validators: {
      onChange: ({ value }) => {
        const password = form.getFieldValue('password')
        return value !== password ? 'Passwords do not match' : undefined
      }
    }
  }}
/>`} language='tsx' />

## Context-based select field integration

The composed select field pattern uses React Context to access form state. This approach is ideal for applications with multiple forms and reusable select components.

## Prerequisites

<CodeHighlight code={`npx shadcn@latest add "https://shuip.plvo.dev/r/tsf-context"`} language='bash' />

## Basic usage

<CodeHighlight code={`import { useAppForm, TanstackForm } from '@/components/ui/tanstack-form/context'

function CountryForm() {
  const form = useAppForm({
    defaultValues: { country: '' },
  })

  return (
    <TanstackForm form={form}>
      <form.AppField name='country'>
        {(field) => (
          <field.SelectField
            options={{
              'United States': 'us',
              'United Kingdom': 'uk',
              'France': 'fr',
            }}
            label='Country'
            placeholder='Select a country'
          />
        )}
      </form.AppField>
      <form.SubmitButton>Submit</form.SubmitButton>
    </TanstackForm>
  )
}`} language='tsx' />

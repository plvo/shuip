---
title: Submit Button
description: Specifically designed for form submission with built-in loading states, making it perfect for forms that need to show submission progress and prevent double submissions.
registryName: submit-button
---

## Built-in features

- **Automatic loading state**: Shows spinner icon when `loading` prop is true
- **Full-width by default**: Optimized for form layouts
- **Customizable icon**: Can override the default loading spinner
- **Form integration**: Native `type="submit"` behavior
